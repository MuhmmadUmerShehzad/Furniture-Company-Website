Q1) Pine Valley Furniture Company Web Solution

Consider the Pine Valley Furniture Company database designed and implemented during the previous semester. Try to recall major relations in the database (CUSTOMER_T, ORDER_T, ORDER_LINE_T, and PRODUCT_T).

There is a need to implement a web based solution for above mentioned database. The web based solution must incorporate following functionalities:

  1. New Customer Registration
  2. Searching of Products
  3. Products Selection and Order Placement
  4. Product Catalog Update
  5. Payment Provision

Tasks:

  1. As a first step design above mentioned (i) to (v) user interfaces using paper and pencil.
  2. Next write appropriate html for creating (i) to (v) user interface prototypes.
  3. Create links for navigation among above pages.
  4. Design a menu using list and styles.
  5. Add help pages for user guidance.
  6. Test user interface prototypes in browser. Either use table for layout or may opt for flow based layout. Experiment with style sheets.

  3. How targeted marketing can be realized?
  4. Add recommendation system feature in your project.
